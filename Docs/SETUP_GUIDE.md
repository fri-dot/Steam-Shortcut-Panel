# Steam Shortcut Panel - Setup Guide

## What You Need

✅ **For running the executable (.exe):**
- Windows 10 or newer
- Python 3.9+ (for first-run setup)
- Internet connection (for pip to install packages)

✅ **For building from source:**
- Windows 10 or newer
- Python 3.9+
- PyInstaller
- Required Python packages (requests, Pillow)

## Quick Start

### Step 1: Build the Executable (One-time setup)

Simply run:
```bash
build_exe.bat
```

This automatically:
- Installs Python dependencies
- Installs PyInstaller
- Generates the blue leaf icon
- Builds the standalone executable

**Wait for the build to complete (may take a few minutes)**

### Step 2: Locate the Executable

After the build completes, find the executable at:
```
dist\Steam Shortcut Panel.exe
```

### Step 3: Run the Application

Double-click `Steam Shortcut Panel.exe`

**First Run:**
- The app checks for required Python packages
- If any are missing, a setup dialog appears automatically
- Click "Install" and wait for the setup to complete
- The app will then start normally

**Future Runs:**
- The app starts directly without any setup dialogs

## Troubleshooting

### "Setup dialog keeps appearing"
- Ensure Python 3.9+ is installed
- Check that your internet connection works
- Try running with administrator privileges

### "Package installation fails"
- Check your internet connection
- Try running the setup dialog again
- If persistent, install manually: `pip install requests Pillow`

### "Icon not showing on shortcuts"
- This is normal if the game doesn't have .ico files
- The app will download from Steam CDN instead
- If CDN also fails, shortcuts will work but without icons

### "Steam library not detected"
- If auto-detection fails, use the "Browse" button to select manually
- Usually at `C:\Program Files\Steam` or `C:\Program Files (x86)\Steam`

## Technical Details

### How the Setup Works

1. **Imports:** The setup module is included in the .exe
2. **Detection:** On startup, Python packages are checked
3. **Installation:** If missing, pip automatically installs them
4. **UI:** A modern dark-themed dialog guides the user
5. **Completion:** After setup, the main app launches

### Icon Details

- **Format:** ICO (Windows icon format)
- **Design:** Blue leaf with subtle vein details
- **Size:** 256x256 pixels
- **Color:** Bright blue (#00a4ff) with darker vein lines
- **Generated by:** `create_icon.py`

### Build Process

The `build_exe.bat` script:
1. Upgrades pip
2. Installs requirements
3. Installs PyInstaller
4. Runs `create_icon.py` to generate the icon
5. Runs PyInstaller with spec file configuration
6. Creates the final executable in `dist/`

### PyInstaller Configuration

The `main.spec` file controls:
- Executable name: "Steam Shortcut Panel"
- Icon file: "icon.ico"
- Window mode: Windowed (no console)
- Hidden imports: requests, PIL, tkinter
- Output: Single-file executable

## Advanced Options

### Rebuild with Different Icon
1. Edit `create_icon.py` to customize the icon
2. Run `python create_icon.py`
3. Rebuild with `build_exe.bat`

### Manual Build
```bash
python create_icon.py
pyinstaller main.spec
```

### Build with Console Window (Debugging)
Edit `main.spec`:
- Change `console=False` to `console=True`
- Rebuild with PyInstaller
