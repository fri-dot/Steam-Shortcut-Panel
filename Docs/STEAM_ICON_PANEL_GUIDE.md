## 🚀 Build Process Breakdown

### What `build_exe.bat` Does:

```
build_exe.bat
    ↓
1. Upgrade pip
    ↓
2. Install requirements
    - requests
    - Pillow
    ↓
3. Install PyInstaller
    ↓
4. Generate icon (create_icon.py)
    - Creates icon.ico
    ↓
5. Build executable (PyInstaller)

    - Compiles Python code
    - Bundles dependencies
    - Embeds blue leaf icon
    ↓
dist/Steam Icon Panel.exe ✓
```

## 🔧 Manual Build (Alternative)

If you prefer to build step-by-step:

```bash
# 1. Install dependencies
python -m pip install -r requirements.txt

# 2. Install PyInstaller
python -m pip install pyinstaller

# 3. Generate icon
python create_icon.py

# 4. Build executable
pyinstaller --onefile --windowed --name "Steam Icon Panel" --icon icon.ico main.py
```

Result: `dist/Steam Icon Panel.exe`

## 🐛 Troubleshooting

### Build Issues

**"build_exe.bat fails"**
- Ensure Python 3.9+ is installed
- Check internet connection
- Run: `python -m pip install -r requirements.txt`
- Try again

**"PyInstaller not found"**
- Run: `python -m pip install pyinstaller`
- Try build again

**"Icon not generating"**
- Check `icon.ico` exists after build starts
- Verify `create_icon.py` has no errors
- Manually run: `python create_icon.py`

### Runtime Issues

**"Setup wizard doesn't appear"**
- This is normal! Packages may already be installed
- Just use the app normally

**"Packages fail to install in setup wizard"**
- Check internet connection
- Try clicking "Cancel" then "Install & Continue" again
- Update Python: `python -m pip install --upgrade pip`

**"App won't start after setup"**
- Try running exe again
- Restart computer
- Reinstall by deleting and re-running setup

## 💡 Tips & Tricks

### Customize the Icon
1. Edit `create_icon.py` to change colors/design
2. Run: `python create_icon.py`
3. Rebuild with: `build_exe.bat`

### Debug Issues
1. Edit `main.spec` - change `console=False` to `console=True`
2. Rebuild to see console output
3. Review error messages

### Optimize Size
- Default: 15-50 MB
- Use UPX compressor in PyInstaller for smaller size
- Edit `build_exe.bat` to add: `--upx-dir=path/to/upx`
